## ğŸ› ï¸ Installation & Setup

1. Clone or download the repository to your local machine
2. Open the project folder in your preferred code editor
3. Open `index.html` in your web browser to view the site
4. For development:
   - Install Sass preprocessor: `npm install -g sass`
   - Compile SCSS to CSS: `sass assets/scss/main.scss assets/css/main.css`
   - Watch for changes: `sass --watch assets/scss/main.scss assets/css/main.css`
